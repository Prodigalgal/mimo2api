# MiMo2API

TypeScript/Node.js OpenAI-compatible gateway for Xiaomi MiMo AI Studio.

The v3 runtime supports Chat Completions, the Responses API family, text/image/file inputs, validated function calls, SQLite persistence, staggered account renewal, temp-mail configuration and VLESS/sing-box proxy management. Anthropic, audio models, TTS, ASR and legacy batch state are intentionally removed.

See [README.md](./README.md) for setup, migration, API and deployment details.
